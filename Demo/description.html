Introduction

This sample app contains multiple samples to demonstrate how use Face API Client Library to make Face APIs calls. Specifically:

  • Face Detection: to detect faces in an image.
    By input a local image, call Detection API to render face rectangles, and get other attributes including gender, age and head-pose.
  • Face Verification: to check two faces belonging to same person or not.
    Calling to verification API to determine whether two faces belong to a same person. By inputting two single-face-images, it returns verification result and confidence.
  • Face Grouping: to group faces by visual similarity.
    Calling to grouping API to divide faces to groups, and each group contains the face with visual similarity. This sample firstly detect faces from the images in a folder then groups the faces and shows the results.
  • Finding Similar Faces: to find similar faces by a given face.
    Slightly different from grouping sample, this sample also searches similar looking faces, but by a input face as a query face, and a list of candidate faces.
  • Face Identification: to identify a specific person from a query face.
    Identification is to tell which person the input face belongs to. The first step is to create a "person database", then the second step is to identify which person the input face belongs to.

Build and Run Sample Project

  1. Click Subscription Key Management to obtain your subscription key which is required for Face API calls.
  2. Open "Sample\FaceApiSamples.sln" in root folder in Visual Studio 2013 Solution.
  3. Put your subscription key string from above step, into file "Sample\MainWindow.xaml.cs", find the line "this.SubscriptionKey = null;" and replace the null value with your key string.
  4. Build Solution and run.
